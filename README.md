#  University Registration System

A simple Java application to manage students and course registrations.

## Features
* **Manage Students:** Add and store student data.
* **Course List:** View and manage available courses.
* **Enrollment:** Register students for specific courses.
* **Organized Code:** Built using Object-Oriented Programming (OOP).

## Tech Stack
* **Language:** Java ☕
* **Build Tool:** Maven 📦

##  File Structure
* `Student.java`: Handles student objects.
* `CourseList.java`: Manages the list of courses.
* `UniversityRegistrationSystem.java`: The main entry point.
* `pom.xml`: Project configuration.

##  How to Run
1. Download or Clone the project.
2. Open it in any Java IDE (IntelliJ, Eclipse, or NetBeans).
3. Run `UniversityRegistrationSystem.java`.
